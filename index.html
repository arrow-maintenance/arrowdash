Mailing list items from the last month

  • Mailing List Summary

The content below is an LLM summary of this month’s activity on the mailing list.

  • Arrow 23.0.1 release and security provenance: Discussion during the 23.0.1 release process led to a new requirement to include SHA256 hashes directly in vote threads to ensure artifact integrity. Thread

  • Proposed IPC format extensions: A proposal was introduced to extend the IPC format by adding an ApplicationData field to messages and introducing an “Empty” message type for specialized signaling. Thread

  • Supporting 24:00:00 in TIME types: Community members discussed whether to allow 24:00:00 in Time32 and Time64 types to improve data fidelity when round-tripping from databases like PostgreSQL and DuckDB. Thread

  • vgi-rpc introduction: Introduction of an open-source RPC framework built on Arrow IPC that provides typed proxies for Python and other languages without a gRPC dependency. Thread

  • Flight SQL Prepared Statement improvements: Proposal to add a boolean field to the protocol indicating whether a prepared statement is an update or a query to help drivers choose the correct execution path. Thread

  • AI-generated contributions and maintenance: Discussion on leveraging LLMs for automated issue triage and a proposal requiring contributors to disclose if they are using AI to generate mailing list responses. Thread

  • IPC file and stream format equivalence: Technical discussion regarding whether IPC files should be strictly guaranteed to be readable as valid IPC streams and how to handle potential schema discrepancies. Thread

  • Arrow’s 10th Anniversary: The community marked ten years since the project’s inception, highlighting its evolution into a universal standard for columnar data exchange. Thread
