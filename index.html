Mailing list items from the last month

  • Mailing List Summary

The content below is an LLM summary of this month’s activity on the mailing list.

  • Release Provenance and Integrity: Following concerns about the canonical nature of release artifacts, the project is moving toward a standard of including SHA512 hashes directly in release vote emails to prevent potential artifact substitution. Thread

  • Apache Arrow 10th Anniversary: The project celebrated a decade of development, marking 10 years since the first commit in February 2016 and its growth into a universal standard for columnar data exchange. Thread

  • Proposed Support for 24:00:00 Time: Developers are re-evaluating the range of the TIME type to support “24:00:00” for better interoperability and fidelity when round-tripping data from databases like PostgreSQL. Thread

  • Introduction of vgi-rpc: A new typed RPC framework for Python was introduced that uses Arrow IPC exclusively, bypassing gRPC dependencies and using Python type annotations to automatically infer schemas. Thread

  • IPC Format Extensions: Discussion is underway to extend the IPC format with ApplicationData on messages and a new Empty message type to provide better metadata handling and stream flexibility. Thread

  • Deprecation of Gandiva and Feather V1: Proposals were made to deprecate pyarrow.gandiva and the legacy Feather V1 reader/writer to reduce maintenance burden on components with low usage or better-maintained alternatives. Thread

  • Equivalence of IPC File and Stream Formats: A technical discussion explored whether IPC files should be guaranteed to be readable as valid streams by simply skipping the header, highlighting discrepancies in how schema metadata is stored in footers. Thread

  • Map Type Field Naming Consistency: Implementers discussed whether field names for Map types (entries, key, value) should be strictly enforced across all language bindings to ensure seamless cross-implementation integration. Thread

  • Release 23.0.1 and Major Rust/Go Updates: The project finalized several releases, including the Arrow 23.0.1 patch release and significant version bumps for the Rust (58.0.0) and Go (18.5.2) implementations. Thread
