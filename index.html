Mailing list items from the last month
Date Subject
  • Mailing List Summary

The content below is an LLM summary of this month’s activity on the mailing list.

  • Apache Arrow 23.0.0 release: The community announced the release of version 23.0.0, which includes 336 resolved GitHub issues across the project’s various language implementations. Thread

  • AI-generated contribution guidelines: A proposal was introduced to establish clear documentation regarding AI-assisted pull requests, emphasizing that contributors must fully understand and own the generated code to reduce maintainer burden. Thread

  • New IPC message type for opaque bytes: Developers are discussing the addition of an OpaqueBytes or ApplicationData message type to the IPC stream format to allow arbitrary metadata or signaling to be interleaved with record batches. Thread

  • Security model for Arrow formats: A draft document has been proposed to formalize security considerations for the Arrow in-memory and IPC formats, with plans to extend these guidelines to specific implementations like C++. Thread

  • Validity buffer requirements for nested types: There is an ongoing debate regarding the specification of nullability in StructArray, specifically whether non-nullable child fields should be required to omit or include validity buffers when the parent struct is null. Thread

  • Arrow ADBC 22 release: The community released version 22 of the Arrow Database Connectivity (ADBC) libraries, providing a standardized database access abstraction for analytical applications. Thread

  • Late materialization in Parquet: Discussion continues on providing lower-level Parquet reader APIs that would allow for more efficient filter pushdown by evaluating expressions directly on encoded data before converting to Arrow format. Thread

  • Java Dataset API roadmap: Users and maintainers discussed the current experimental status of the Java Dataset API and identified the need for a roadmap toward production readiness and feature parity with the C++ implementation. Thread

  • Rust implementation maintenance: Several patch releases were coordinated for the Rust implementation (arrow-rs), including version 57.3.0 to address specific bugs and improve documentation builds for the object store component. Thread
