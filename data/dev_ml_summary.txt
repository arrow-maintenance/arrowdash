* **Java JDK 17 requirement:** The community reached a consensus to set JDK 17 as the minimum requirement for Arrow Java starting with version 20.0.0. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Java%20Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **TIME type and 24:00:00 support:** Discussion regarding the current TIME type's inability to represent "24:00:00," which causes data loss when round-tripping data from databases like PostgreSQL and DuckDB. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:TIME%20support%20for%20240000)

* **Flight SQL Prepared Statement execution:** Proposal to improve how Flight SQL clients determine whether to use query or update paths, potentially adding a boolean indicator to avoid incorrect heuristics in drivers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Flight%20SQL%20Indicating%20execution%20path%20for%20Flight%20SQL%20Prepared%20Statements)

* **Java 19.0.0 release:** Apache Arrow Java 19.0.0 was officially released, featuring various improvements and bug fixes for the Java implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Java%20Release%20Apache%20Arrow%20Java%2019.0.0%20RC1)

* **ADBC driver migration:** Progress update on moving BigQuery, Databricks, and Snowflake drivers from the main ADBC repository to a dedicated "adbc-drivers" organization for better maintenance. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20drivers%20contrib%20org)

* **LLMs for project maintenance:** Exploration of using Large Language Models to automate repository management tasks, such as triaging issues and applying component labels. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20LLMs%20to%20aid%20with%20maintenance)

* **Java Flight UDS regression:** Identification of a critical bug in Java Flight versions 16-18 where using Unix Domain Sockets with Epoll triggers a ClassCastException. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Java%20Flight%20Critical%20Regression%20in%20v16-v18%20FlightClient%20UDS%20throws%20ClassCastException)

* **Rust implementation releases:** The Rust implementation officially released version 58.0.0 followed quickly by a 58.1.0 patch release. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RESULT%20VOTE%20RUST%20Release%20Apache%20Arrow%20Rust%2058.0.0%20RC1)

* **GSoC 2026 participation:** Maintainers clarified that the Apache Arrow project does not currently plan to participate in the 2026 Google Summer of Code program. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Apache%20Arrow%20participation%20in%20GSoC%202026)

* **Map type field naming:** Technical discussion on whether field names for Map types (entries, key, value) should be strictly enforced across different language implementations for better interoperability. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Map%20type%27s%20key%2Fitem%2Fvalue%20field%20names)
