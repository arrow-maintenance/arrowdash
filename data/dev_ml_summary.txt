* **Erlang implementation donation:** The community voted to accept the donation of an Erlang implementation (built on Rust bindings), with the repository now being transferred to the official Apache organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **ADBC Partitioned Bulk Ingest API:** A new proposal introduces a write-side mirror to the partitioned-read API to support distributed parallel writing for engines like Spark and Flink. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Parquet Bloom Filter SIMD optimizations:** Significant performance improvements for C++ Parquet Bloom Filter probes were implemented using AVX2 and xsimd, showing up to 5x speedups in-cache. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Canonical Extension Type for ranges:** Discussion on adding `arrow.range` to represent bounded mathematical intervals with specific metadata for inclusive/exclusive bounds. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Automated GitHub Copilot reviews:** To address limited reviewer bandwidth, the project enabled automated Copilot reviews via `.asf.yaml` to catch trivial problems before human review. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **IBM Db2 support:** Engineers from IBM proposed adding Arrow support for Db2, starting with a new ADBC driver developed under the `adbc-drivers` Foundry. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **Apache Arrow .NET 23.0.0 release:** The community announced the release of version 23.0.0 for the .NET implementation, including various updates for Flight and NuGet packages. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Apache%20Arrow%20.NET%2023.0.0%20released)

* **UTF8 metadata restrictions:** Discussion on whether to strictly enforce UTF8 for field and schema metadata in C++ and IPC implementations to ensure compatibility with Flatbuffers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8%3F)

* **Rust patch releases:** Several maintenance releases (58.3.0, 57.3.1, 56.2.1) were coordinated to backport fixes for potential offset overflow issues in the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Planned%20patch%20releases%20this%20week)
