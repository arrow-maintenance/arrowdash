* **Donation of pyarrow-stubs:** The community voted to accept the donation of the pyarrow-stubs codebase, which provides PEP 484 type annotations for PyArrow, ensuring they stay in sync with the main library. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:PYTHON%20Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **Apache Arrow 24.0.0 Release:** This major release includes 259 resolved issues, featuring improvements across multiple language bindings and resolving specific date/time testing failures on macOS. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Apache%20Arrow%2024.0.0%20released)

* **Flight SQL Prepared Statement Improvements:** Proposal and vote to add an `is_update` boolean field to `ActionCreatePreparedStatementResult` so clients can correctly route prepared statements without using string-parsing heuristics. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Flight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Dictionary-encoded extension types in IPC:** Discussion on addressing a limitation in the IPC format that prevents distinguishing between dictionary-encoded extension types and extension types with dictionary storage. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **Canonical BigDecimal Extension Type:** Proposal for a new canonical extension type to support arbitrary-precision decimals with variable scales, facilitating high-fidelity data transfer from engines like PostgreSQL and Trino. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Format%20Proposal%20for%20a%20Canonical%20BigDecimal%20Extension%20Type)

* **IBM Db2 Arrow Support:** Engineers from IBM initiated a discussion on adding native Arrow support for Db2, with the community guiding the development of an ADBC driver under the adbc-drivers organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20DB2%20Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **Rust Security Policy:** A proposed security policy for the Rust implementations of Arrow and Parquet aims to clarify the distinction between standard bugs and security vulnerabilities. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RUST%20Discuss%20Security%20Policy)

* **ADBC Logo Proposal:** Discussion on a new visual identity for the ADBC subproject to differentiate it from the core Arrow project while maintaining brand consistency. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Logo%20Proposal)

* **Arrow Erlang IP Clearance:** Progress on the donation of the Erlang implementation, including the preparation of Software Grant Agreements and Individual Contributor License Agreements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Erlang%20Arrow%20Erlang's%20Grant%20Documents)

* **Apache Arrow .NET 23.0.0 Release:** Announcement of the latest .NET release, including updated packages for Flight SQL and new PowerShell-based verification scripts for Windows users. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Apache%20Arrow%20.NET%2023.0.0%20released)
