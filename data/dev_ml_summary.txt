* **Apache Arrow 24.0.0 Release:** The community successfully voted on and released version 24.0.0, which includes over 250 resolved issues and improvements across the project. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **pyarrow-stubs donation:** The project approved the donation of the independently maintained pyarrow-stubs codebase, integrating type annotations directly into the Arrow repository to better support the Python community. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20PYTHON%20Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **Canonical BigDecimal Extension Type:** A proposal was introduced to create a canonical extension type for BigDecimal to support arbitrary-precision decimals with variable scales, facilitating high-fidelity data transport from engines like PostgreSQL and Trino. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Format%20Proposal%20for%20a%20Canonical%20BigDecimal%20Extension%20Type)

* **LargeMap format addition:** Discussion on adding a LargeMap type that supports 64-bit offsets, enabling the encoding of map data that exceeds the 32-bit offset limit of the standard Map type. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20LargeMap%20format%20addition)

* **Apache Arrow ADBC 23 Release:** The Arrow Database Connectivity (ADBC) subproject released version 23, featuring 41 bug fixes and enhancements for database abstraction. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)

* **IPC Dictionary-Encoded Extension Types:** Developers are addressing a limitation in the IPC schema serialization that prevents distinguishing between dictionary-encoded extension types and extension types with dictionary storage. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **Flight SQL Prepared Statement Updates:** A vote was held to add an `is_update` field to `ActionCreatePreparedStatementResult` to help clients correctly identify the execution path for DML/DDL statements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Flight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Rust Security Policy:** The Rust implementation community is formalizing a security policy for `arrow-rs` and `parquet-rs` to clarify the triage process for security vulnerabilities versus standard bugs. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RUST%20Discuss%20Security%20Policy)

* **Go Implementation Release 18.6.0:** The Go subproject released version 18.6.0, including a fix for release verification scripts and updated dependencies. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Go%20Release%20Apache%20Arrow%20Go%2018.6.0%20RC0)

* **ADBC Logo Proposal:** A new logo for the ADBC subproject was proposed to provide a distinct identity while remaining clearly within the Apache Arrow visual family. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20ADBC%20Logo%20Proposal)
