* **Arrow 24.0.0 release candidate:** Voting is underway for the next major release, featuring over 250 resolved issues across the various language implementations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **Numba support for PyArrow:** Discussion on integrating Numba JIT support directly into PyArrow to enable high-performance scientific calculations on Arrow data while avoiding C++ ABI stability issues. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Numba%20support%20for%20PyArrow)

* **Java JDK 17 minimum requirement:** The community reached a consensus to set JDK 17 as the minimum required version for the upcoming Arrow Java 20.0.0 release. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **Donation of pyarrow-stubs:** The project voted to accept the donation of the pyarrow-stubs codebase, which will bring official PEP 484 type hint support directly into the main repository. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **C Device Data Interface stability:** Developers discussed why the C Device Data Interface remains marked as experimental, citing a need for more non-CUDA usage and potential future requirements for an asynchronous stream interface. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Experimentality%20of%20Arrow%20C%20Device%20Data%20Interface)

* **Flight SQL is_update field:** A proposal and vote to add an `is_update` boolean to prepared statement results to help clients correctly choose between query and update execution paths. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **LargeMap format addition:** Proposal to add a `LargeMap` type to the Arrow specification to support 64-bit offsets, ensuring parity with other "Large" types used by libraries like Polars. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:LargeMap%20format%20addition)

* **Dictionary-encoded extension types in IPC:** Discussion on a refinement to the IPC format to better distinguish between dictionary-encoded extensions and extension types that use a dictionary as their storage type. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **ADBC Release 23:** The project announced the release of ADBC version 23, including 41 bug fixes and improvements across C++, Go, Java, Python, and Rust components. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)
