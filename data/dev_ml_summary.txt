* **Apache Arrow 24.0.0 release:** The community approved and released version 24.0.0, resolving over 250 issues across multiple language implementations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **Numba support for PyArrow:** Discussion on integrating a Numba-based JIT compiler layer into PyArrow to enable high-performance scientific calculations directly on Arrow data. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DiscussPython%20Numba%20support%20for%20PyArrow)

* **pyarrow-stubs donation:** The community voted to accept the donation of the `pyarrow-stubs` codebase to provide official PEP 484 type stub annotations for the project. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSPYTHON%20Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **Canonical BigDecimal extension type:** Proposal for a new canonical extension type for arbitrary-precision decimals to improve data integrity when moving data between Arrow and engines like PostgreSQL or Trino. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSFormat%20Proposal%20for%20a%20Canonical%20BigDecimal%20Extension%20Type)

* **LargeMap format addition:** Discussion on adding a `LargeMap` type with 64-bit offsets to the Arrow specification to support map data structures that exceed the 32-bit offset limit. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20LargeMap%20format%20addition)

* **JDK 17 minimum for Java:** The community reached consensus to set JDK 17 as the minimum requirement for the Arrow Java implementation starting with version 20.0.0. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSJava%20Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **Flight SQL Prepared Statement execution:** Proposal to add an `is_update` field to `ActionCreatePreparedStatementResult` so servers can signal the correct execution path for queries versus updates. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSFlight%20SQL%20Indicating%20execution%20path%20for%20Flight%20SQL%20Prepared%20Statements)

* **IPC dictionary-encoded extension types:** Investigation into a limitation in the IPC schema serialization that prevents distinguishing between dictionary-encoded extension types and extension types with dictionary storage. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **Apache Arrow ADBC 23 release:** Release of version 23 for Arrow Database Connectivity (ADBC) libraries, including major updates for C#, Java, and Rust subcomponents. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)

* **Rust Security Policy:** Discussion on establishing formal security reporting guidelines for the Arrow-RS and Parquet-RS crates to improve vulnerability management. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RUST%20Discuss%20Security%20Policy)
