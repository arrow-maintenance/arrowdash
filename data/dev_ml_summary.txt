* **Rust patch releases:** Andrew Lamb coordinated patch releases for versions 58.3.0, 57.3.1, and 56.2.1 of the Rust implementation to backport critical offset overflow fixes. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RUST%20Planned%20patch%20releases%20this%20week)

* **Hackathon at Community Over Code Glasgow 2026:** The community is organizing Arrow's participation in the upcoming October hackathon, seeking mentors and defining task lists for new contributors. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Hackathon%20at%20Community%20Over%20Code%20Glasgow%202026%20%E2%80%94%20interest%20from%20Arrow%3F)

* **Apache Arrow .NET 23.0.0 release:** The .NET implementation has released version 23.0.0, including updates to core packages, Flight, and SQL operations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Apache%20Arrow%20.NET%2023.0.0%20released)

* **Metadata UTF8 enforcement:** A discussion was raised regarding whether the C++ implementation and the C Data Interface should strictly enforce UTF8 for field and schema metadata to ensure specification compliance. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Field/schema/custom%20metadata%20restriction%20to%20UTF8%3F)

* **AVX2 SBBF probe for Parquet:** Proposal to introduce AVX2-accelerated Split Block Bloom Filter (SBBF) probes in C++ to improve Parquet read performance, achieving significant speedups on in-cache and DRAM-heavy workloads. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20AVX2%20SBBF%20probe%20for%20parquet/bloom_filter.cc)

* **Erlang implementation donation:** The community voted to accept the donation of the `arrow-erlang` library, moving the codebase into the official Apache Arrow repository. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **ADBC Partitioned Bulk Ingest API:** A new proposal aims to add partitioned bulk ingestion to ADBC, enabling distributed engines like Spark to perform parallel writes with atomic commits. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Canonical Range Extension Type:** Discussion on adding a canonical `arrow.range` extension type to represent mathematical intervals, supporting inclusive/exclusive boundaries and unbounded endpoints. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Automated GitHub Copilot reviews:** The project enabled automated Copilot reviews for pull requests to reduce the burden on human maintainers by catching trivial issues earlier in the process. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **Preparing Arrow Java 20.0.0:** Triaging has begun for the upcoming Java 20.0.0 release, with focus on organizing GitHub issues and managing milestone targets. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:HEADS%20UP%20Preparing%20Arrow%20Java%2020.0.0)
