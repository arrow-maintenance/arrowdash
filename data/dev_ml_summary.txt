* **Variant type implementation:** Developers are coordinating the implementation of the Variant type in Arrow C++, focusing on high-performance encoding and compatibility with the existing Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **CI resource optimization:** In response to ASF Infrastructure limits, the project successfully reduced its GitHub Actions runner minute consumption by nearly 30% through workflow streamlining and efficiency improvements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **DuckDB ADBC extension:** A proposal to donate a new DuckDB extension for ADBC prompted discussion on ecosystem collaboration and comparisons with existing community-maintained drivers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Extension%20for%20DuckDB)

* **Arrow Erlang donation:** The donation of the Erlang implementation is complete, with the repository now officially transferred to the Apache Software Foundation organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RESULT%20VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **AI-assisted PR reviews:** The community is establishing guidelines for GitHub Copilot automated reviews, including disabling them for draft PRs to minimize noise for contributors. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **Flight SQL update flag:** A vote passed to add an `is_update` boolean to Flight SQL’s prepared statement results to help clients distinguish between data modifications and queries.
