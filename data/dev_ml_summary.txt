* **Erlang implementation donation:** The community voted to accept the donation of an Erlang implementation of Apache Arrow, which is currently based on bindings to the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **ADBC partitioned bulk ingest API:** Proposal for a new API to support parallel atomic writes from distributed systems like Spark or Flink using opaque handle and receipt tokens. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **SIMD optimizations for Parquet Bloom Filters:** Discussion and subsequent pull request regarding the use of AVX2 and NEON SIMD instructions to accelerate Split Block Bloom Filter probes in C++. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Canonical extension type for bounded ranges:** Proposal to add an `arrow.range` type for representing mathematical intervals, standardizing how ranges are exchanged between Arrow-compatible analytical libraries. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **IBM Db2 integration support:** IBM engineers proposed adding Apache Arrow support for IBM Db2, starting with a new ADBC driver and evaluating Flight SQL for high-performance data exchange. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **UTF-8 metadata enforcement:** Discussion on whether the C++ implementation should strictly validate field and schema metadata as UTF-8 to ensure compliance with the IPC format and Flatbuffers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8%3F)

* **Flight SQL dialect-related SqlInfo codes:** Proposal to add new codes that allow Flight SQL backends to communicate specific SQL dialect capabilities, such as supported limit/offset grammar and null ordering. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20dialect-related%20SqlInfo%20codes)

* **Automated GitHub Copilot reviews trial:** The project is trialing automated Copilot-generated code reviews on pull requests to help identify trivial issues and reduce maintainer overhead. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **Apache Arrow .NET 23.0.0 release:** Announcement of the latest .NET release, including updated packages for Flight SQL, operations, and compression. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Apache%20Arrow%20.NET%2023.0.0%20released)
