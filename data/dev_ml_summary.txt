* **Erlang implementation donation:** The community voted to accept the donation of an Erlang implementation of Apache Arrow, which currently provides bindings to the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **ADBC partitioned bulk ingest:** Proposal for a new API to support coordinated, parallel data ingestion from distributed writers like Spark or Flink into RDBMS or table formats like Iceberg. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Metadata UTF-8 enforcement:** Discussion on whether the Arrow specification should strictly enforce UTF-8 for field and schema metadata to prevent issues with Flatbuffers serialization and improve cross-language interoperability. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8%3F)

* **IBM Db2 support:** IBM engineers proposed adding Arrow support for Db2 via ADBC and Flight SQL, with initial work focused on a community-maintained ADBC driver. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSADBCDB2%20Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **Parquet Bloom filter optimization:** Proposal to implement AVX2-accelerated Split Block Bloom Filter (SBBF) probes in the C++ implementation using xsimd to significantly improve read performance. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Flight SQL dialect metadata:** Proposal to add new SqlInfo codes to Flight SQL to help clients handle backend-specific SQL dialects, such as LIMIT/OFFSET syntax and boolean literal forms. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSFlightSQL%20Add%20dialect%2Drelated%20SqlInfo%20codes)

* **Python stubs donation:** The `pyarrow-stubs` library has been formally donated to the project to provide official static typing support for Python developers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTEPYTHON%20Donate%20pyarrow%2Dstubs%20to%20Apache%20Arrow)

* **IPC file vs. stream documentation:** Proposed updates to the format documentation to clarify the equivalence and technical differences between the Arrow IPC file and IPC stream formats. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Equivalence%20of%20IPC%20file%20and%20stream%20formats)

* **Flight SQL update indicator:** Discussion and vote on adding an `is_update` field to Flight SQL prepared statements to help drivers automatically determine the correct execution path for DML/DDL queries. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTEFlight%20SQL%20Add%20is%5Fupdate%20field%20to%20ActionCreatePreparedStatementResult)
