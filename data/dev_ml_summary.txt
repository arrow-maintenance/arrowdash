* **GSoC 2026 participation:** Maintainers clarified that the project does not currently plan to participate in Google Summer of Code 2026, consistent with previous years. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Apache%20Arrow%20participation%20in%20GSoC%202026)

* **TIME support for 24:00:00:** Discussion on whether to revisit the exclusion of `24:00:00` in the TIME type to improve data fidelity with databases like PostgreSQL and DuckDB. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:TIME%20support%20for%2024%2000%2000)

* **Flight SQL Prepared Statements execution:** Proposal to consolidate execution modes and add a boolean field to indicate whether a statement follows a query or update path for better driver compatibility. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Indicating%20execution%20path%20for%20Flight%20SQL%20Prepared%20Statements)

* **Java Flight UDS Regression:** A critical bug was identified in Java Flight (v16–v18) where `FlightClient` fails to initialize with Unix Domain Sockets due to an incorrect cast in Netty/gRPC logic. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Critical%20Regression%20in%20v16-v18%20FlightClient%20UDS%20throws%20ClassCastException)

* **Map type field naming:** Discussion on whether the Arrow specification enforces field names ("entries", "key", "value") for Map types, concluding that names are common but not strictly enforced to maintain compatibility with formats like Parquet. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Map%20type%27s%20key%20item%20value%20field%20names)

* **LLM-aided maintenance:** Exploration of using Large Language Models to automate repo management tasks, such as adding component labels to unlabelled issues. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:LLMs%20to%20aid%20with%20maintenance)

* **Arrow Java 19.0.0 and JDK 17:** Following the successful release of Arrow Java 19.0.0, the community reached consensus to raise the minimum required JDK to version 17 for the upcoming 20.0.0 release. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **ADBC driver migration:** Update on migrating BigQuery, Databricks, and Snowflake drivers from the main ADBC repository to a dedicated `adbc-drivers` organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20drivers%20contrib%20org)

* **Rust Object Store and Arrow-rs releases:** Both the Apache Arrow Rust Object Store 0.13.2 and the core Arrow Rust 58.1.0 implementations were successfully voted on and released. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Rust%2058.1.0%20RC1)
