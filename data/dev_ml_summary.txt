* **Donate Apache Arrow Erlang:** The community approved the donation of the Erlang implementation, which has now been officially transferred to the Apache Arrow organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **Canonical Range Extension Type:** Proposal to add a canonical extension type for representing mathematical intervals (bounded ranges), with discussion on handling various boundary closedness and PostgreSQL compatibility. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **ADBC Partitioned Bulk Ingest:** Discussion on a new API for ADBC to support distributed, multi-connection writes with atomic commit/abort semantics for RDBMS and table formats like Iceberg. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Automated GitHub Copilot Reviews:** Trial of AI-driven PR reviews to reduce maintainer burden, including feedback on managing clutter for draft PRs and documenting expectations for contributors. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **C++ Variant Type Support:** Collaboration is underway to implement a high-performance Variant type in C++ to support flexible, JSON-like data structures within Arrow. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **Conbench Benchmarking Infrastructure:** Plans were shared to develop "Conbench v2," a refactor of the project's benchmarking orchestration and database to improve stability and reduce infrastructure costs. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Status%20of%20Arrow%20conbench%20data%20%2B%20conbench%20OSS%20project)

* **Parquet Bloom Filter Optimization:** Significant performance improvements for Parquet SBBF probes were proposed via SIMD autovectorization and xsimd kernels. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet/bloom_filter.cc)

* **GitHub Actions Resource Optimization:** In response to high usage of ASF shared runners, the community is triaging workflows to reduce build times and job concurrency. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **Pyarrow-stubs Donation:** The software grant for the pyarrow-stubs library has been formally filed and published, concluding its move into the Apache Arrow project. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20pyarrow-stubs%20to%20Apache%20Arrow)
