* **Arrow 24.0.0 release:** The community released version 24.0.0, which includes over 250 resolved issues across the various language implementations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **Donation of pyarrow-stubs:** Following a successful vote, the `pyarrow-stubs` codebase will be donated to the project to provide official PEP 484 type annotations for PyArrow. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **ADBC 23 release:** Apache Arrow ADBC version 23 was released, featuring 41 resolved issues across C/C++, Go, Python, Java, and other supported drivers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)

* **Canonical BigDecimal extension type:** A proposal was introduced to create a canonical extension type for arbitrary-precision decimals to support variable scales per row, as required by engines like PostgreSQL and Trino. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20for%20a%20Canonical%20BigDecimal%20Extension%20Type)

* **IPC dictionary-encoded extension types:** Developers discussed updating the IPC serialization format to better distinguish between dictionary-encoded extension types and extension types that use dictionary storage. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **LargeMap format addition:** Discussion regarding the addition of a `LargeMap` type to the Arrow specification to support 64-bit offsets for map data, matching the pattern used for `LargeList` and `LargeString`. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:LargeMap%20format%20addition)

* **Flight SQL prepared statement metadata:** A vote was held to add an `is_update` field to the `ActionCreatePreparedStatementResult` to help clients correctly identify the execution path for prepared statements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **ADBC logo proposal:** The community is reviewing a proposed official logo for the ADBC subproject to provide a distinct visual identity while maintaining brand consistency with Apache Arrow. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Logo%20Proposal)

* **Arrow Rust 58.2.0 release:** Version 58.2.0 of the Rust implementation was released, including various improvements and bug fixes to the core crates and Parquet support. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Rust%2058.2.0%20RC1)

* **Arrow Go 18.6.0 release:** The Go implementation version 18.6.0 was approved and released, addressing minor verification issues discovered during the release candidate phase. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Go%2018.6.0%20RC0)
