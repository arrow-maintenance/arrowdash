* **Erlang repository donation:** The Erlang implementation has been officially donated to the project and transferred to the Apache GitHub organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **Conbench v2 redesign:** Wes McKinney proposed a major update to the Conbench benchmarking infrastructure, featuring a Go backend and TypeScript frontend to improve performance and reliability. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Status%20of%20Arrow%20conbench%20data%20%2B%20conbench%20OSS%20project)

* **Bounded range extension type:** Discussion continues regarding the canonical `arrow.range` type, focusing on naming conventions and the complexities of handling open versus closed bounds in columnar compute functions. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **CI resource optimization:** In response to ASF-wide capacity limits, the project successfully reduced its GitHub Actions runner usage by approximately 27% through workflow streamlining. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **Variant type implementation:** Developers are coordinating the implementation of the Variant type in C++, emphasizing the need for full spec compliance and native performance to support downstream projects like Iceberg-cpp. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **GitHub Copilot review guidelines:** The community agreed to disable automated Copilot reviews for draft PRs and is establishing formal expectations for how contributors should address AI-generated review comments. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **DuckDB ADBC extension donation:** Columnar-tech proposed donating a DuckDB extension for ADBC to the ASF, sparking a debate on governance and the comparison between hand-coded and AI-assisted community extensions. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Extension%20for%20DuckDB)

* **Rust implementation releases:** The project approved and published the releases of Apache Arrow Rust 59.0.0 and the Arrow Rust Object Store 0.14.0. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Rust%2059.0.0%20RC2)
