* **SBBF Bloom Filter Performance:** Dan Mattheiss implemented a branchless autovectorized probe for Parquet Split Block Bloom Filters, achieving significant performance gains on NEON and AVX2 across major compilers without using manual intrinsics. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Canonical Range Type Proposal:** Discussion is underway for adding `arrow.range` and `arrow.range_inc` canonical extension types to represent mathematical intervals, supporting both uniform and per-value closedness for better interoperability with pandas and PostgreSQL. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Automated GitHub Copilot Reviews:** The community is experimenting with automated GitHub Copilot PR reviews to catch trivial issues early, while refining policies on when to disable it (e.g., for draft PRs) and how contributors should address AI feedback. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **Erlang Implementation Donation:** The donation process for the Erlang implementation is nearly complete, with the repository successfully transferred to the Apache Software Foundation's infrastructure as `arrow-erlang`. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RESULTVOTE%20Donate%20Apache%20Arrow%20Erlang)

* **Conbench Redesign and Infrastructure:** Following a migration to new infrastructure, Wes McKinney proposed developing a "v2" of Conbench to improve the stability, performance, and maintainability of the project's continuous benchmarking system. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Status%20of%20Arrow%20conbench%20data%20%2B%20conbench%20OSS%20project)

* **Variant Type Support in C++:** Coordination is happening to implement the Variant type in Arrow C++, utilizing the existing Rust implementation as a reference to ensure cross-language compatibility and native performance. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **CI Resource Optimization:** Following reports of high consumption of ASF shared GitHub-hosted runner resources, the community is working on streamlining workflows and reducing the number of concurrent or redundant CI jobs. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **Arrow Rust 59.0.0 Release:** The Apache Arrow Rust implementation version 59.0.0 was released, featuring numerous updates across the crate ecosystem including core Arrow and Parquet components. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTERUST%20Release%20Apache%20Arrow%20Rust%2059.0.0%20RC2)

* **ADBC Office Hours and Flight SQL Updates:** The project hosted an ADBC community office hours session and voted to add an `is_update` field to `ActionCreatePreparedStatementResult` in Flight SQL to better support DDL and DML operations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20ADBC%20Office%20Hours%20on%20June%2011)
