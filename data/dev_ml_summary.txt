* **Donation of Erlang implementation:** The community voted to accept a new Erlang implementation of Apache Arrow, which is currently based on bindings to the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **ADBC Partitioned Bulk Ingest API:** Proposal for a new API to support parallelized data ingestion, allowing distributed writers like Spark or Flink to coordinate atomic commits across multiple connections. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Arrow Range extension type:** Discussion on adding a canonical extension type to represent bounded intervals (e.g., `[0, 10)`), with debate on how to handle inclusive/exclusive bounds and PostgreSQL compatibility. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Parquet Bloom Filter SIMD optimization:** Proposal to use xsimd and branchless autovectorization to significantly speed up Split Block Bloom Filter (SBBF) probes in the C++ implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Automated GitHub Copilot reviews:** Maintainers have enabled automated AI-powered PR reviews to help identify trivial issues and reduce the manual burden on human reviewers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **Flight SQL Dialect Metadata:** New SqlInfo codes were proposed to help clients identify backend-specific SQL syntax for features like LIMIT/OFFSET and NULLS FIRST/LAST ordering. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20dialect-related%20SqlInfo%20codes)

* **Enforcing UTF8 for metadata:** Discussion on whether to strictly enforce UTF8 for field and schema metadata in C++ and IPC writers to ensure compatibility with Flatbuffers specifications. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8%3F)

* **IBM Db2 ADBC driver:** IBM engineers proposed adding official Arrow support for Db2, starting with a driver implementation in the community-maintained ADBC driver organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **Apache Arrow .NET 23.0.0 release:** The .NET implementation reached version 23.0.0, featuring updated Flight SQL contracts and improved support for column default value metadata. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Apache%20Arrow%20.NET%2023.0.0%20released)
