* **Numba support for PyArrow:** Proposal to integrate a Numba integration layer into PyArrow to enable JIT-compiled scientific calculations directly on Arrow data. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DiscussPython%20Numba%20support%20for%20PyArrow)

* **Arrow Java 19.0.0 release and JDK 17 bump:** Following the release of Java 19.0.0, the community reached consensus to raise the minimum requirement to JDK 17 for version 20.0.0. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTEJava%20Release%20Apache%20Arrow%20Java%2019.0.0%20RC1) | [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSJava%20Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **ADBC 23 release and driver migration:** Release of ADBC version 23 was finalized, coinciding with the migration of BigQuery, Databricks, and Snowflake drivers to a dedicated contrib organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1) | [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20drivers%20contrib%20org)

* **C Device Data Interface and Async Support:** Discussion on moving the C Device Data Interface out of experimental status and exploring an async stream interface for non-CUDA CPU workflows. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Experimentality%20of%20Arrow%20C%20Device%20Data%20Interface)

* **PyArrow-stubs donation:** Proposal to donate and merge the independent `pyarrow-stubs` codebase into the main Arrow repository to provide built-in PEP 484 type annotations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSPYTHON%20Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **Flight SQL Prepared Statements improvement:** Technical discussion on adding a field to `ActionCreatePreparedStatementResult` to help clients determine if a statement is a query or an update. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSFlight%20SQL%20Indicating%20execution%20path%20for%20Flight%20SQL%20Prepared%20Statements)

* **LargeMap format addition:** Proposal to extend the Arrow specification with a `LargeMap` type using 64-bit offsets to prevent casting issues during large-scale data processing. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20LargeMap%20format%20addition)

* **Rust implementation releases:** The community approved releases for both the core Arrow Rust 58.1.0 and the Rust Object Store 0.13.2. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTERUST%20Release%20Apache%20Arrow%20Rust%2058.1.0%20RC1) | [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTERUST%20Release%20Apache%20Arrow%20Rust%20Object%20Store%200.13.2%20RC2)
