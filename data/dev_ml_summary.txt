[Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Rust implementation releases:** The community successfully released versions 59.0.0 and 0.14.0 of the Rust implementation and Rust Object Store, respectively. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20RUST%20Release%20Apache%20Arrow%20Rust%2059.0.0%20RC2)

* **Automated GitHub Copilot reviews:** Debate over the utility of automated AI PR reviews, leading to a decision to restrict them to non-draft PRs to minimize noise for developers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **C++ Variant type support:** Coordination to implement Variant type support in C++ to match functionality in the Rust implementation, which is a key requirement for Iceberg-cpp. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **CI resource optimization:** Following a warning from ASF Infrastructure regarding high runner usage, the project successfully reduced its CI resource consumption by approximately 27%. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **DuckDB ADBC extension:** Discussion regarding the donation of a hand-coded DuckDB ADBC extension to the Arrow project to foster a neutral, collaborative environment for ADBC client development. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Extension%20for%20DuckDB)

* **Deprecating IPC Tensor messages:** Proposal to deprecate legacy experimental Tensor and SparseTensor IPC messages in favor of more modern fixed-shape and variable-shape tensor extension types. * **JSON schema representation:** Proposal to establish a canonical, human-readable JSON format for Arrow schemas to simplify application-level tasks and ADBC metadata handling. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Format%20JSON%20representation%20of%20Arrow%20schemas)
