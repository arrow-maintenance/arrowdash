* **Arrow 24.0.0 release:** The community released version 24.0.0, addressing 259 resolved issues across the project's various language implementations and components. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **ADBC 23 release:** The Arrow Database Connectivity (ADBC) subproject released version 23, which includes over 40 bug fixes and improvements to drivers and the core abstraction layer. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)

* **pyarrow-stubs donation:** The community voted to accept the donation of the pyarrow-stubs codebase, integrating official PEP 484 type annotations into the main Arrow repository for Python users. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **LargeMap format proposal:** Discussion regarding the addition of a `LargeMap` type to the Arrow specification to support 64-bit offsets for map data, accommodating datasets that exceed 2GB. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:LargeMap%20format%20addition)

* **Canonical BigDecimal extension type:** A new proposal for a canonical extension type to support arbitrary-precision decimals with variable scale per row, improving compatibility with database engines like PostgreSQL and Trino. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20for%20a%20Canonical%20BigDecimal%20Extension%20Type)

* **Flight SQL prepared statement updates:** Proposal to add an `is_update` boolean to the `ActionCreatePreparedStatementResult` message to help clients properly handle network flows for DML/DDL statements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **IPC representation of dictionary-encoded extension types:** Developers identified a limitation in the IPC schema that makes it difficult to distinguish between dictionaries of extension types and extension types with dictionary storage. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **Rust security policy:** Discussion and proposal for a formalized security policy for the Arrow-rs and Parquet-rs implementations to better manage vulnerability reporting and disclosure. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Security%20Policy)

* **Async vs Device API experimental status:** A debate on whether to separate the experimental status of the Async API from the Device API to encourage wider adoption and validation of asynchronous C-data workflows. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Experimentality%20of%20Arrow%20C%20Device%20Data%20Interface)

* **Arrow Go 18.6.0 release:** The Go implementation reached version 18.6.0, including a fix for the release verification script regarding the Apache keys URL. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Go%2018.6.0%20RC0)
