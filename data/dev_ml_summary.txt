* **Apache Arrow 23.0.0 release:** The community released version 23.0.0, which includes 336 resolved issues across multiple language implementations and components. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Apache%20Arrow%2023.0.0%20released)

* **Policy on AI-generated contributions:** Discussion on establishing documentation guidelines for LLM-generated pull requests, requiring contributors to fully understand, debug, and own any AI-generated code to prevent maintainer burnout. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20AI-generated%20contributions)

[Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Apache%20Arrow%20ADBC%2022%20Released)

* **Carquet standalone C library:** Introduction of "Carquet," a pure C99 library for reading and writing Parquet files designed for embedded systems and environments without Arrow or C++ dependencies. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Carquet%20-%20standalone%20C%20library%20for%20Parquet%20files)

* **JDBC Driver thread-safety:** Clarification on the concurrency model of the Java JDBC driver, noting that Connections and their associated Statements are not thread-safe and should be used within a single thread. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Java%20JDBC%20Driver%20thread-safety%20guarantees)
