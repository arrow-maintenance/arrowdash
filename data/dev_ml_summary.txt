* **Canonical Range extension type:** Proposal to introduce an `arrow.range` extension type for mathematical intervals, including discussion on inclusive/exclusive bounds and compatibility with PostgreSQL and Pandas. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Automated Copilot PR reviews:** The project is testing automated GitHub Copilot reviews to catch trivial code issues, with discussion focused on managing bot noise in draft pull requests and setting contributor expectations. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **Arrow Erlang donation and transfer:** The community successfully voted to accept the donation of the Erlang implementation, and the repository has been officially transferred to the Apache organization. * **Flight SQL metadata improvement:** A vote passed to add an `is_update` field to prepared statement results, allowing clients to distinguish between data-returning queries and data-modifying updates. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTEFlight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Arrow Java 20.0.0 release preparation:** Maintainers have begun triaging issues for the upcoming Java 20.0.0 release and are seeking community input on must-have features. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Preparing%20Arrow%20Java%2020.0.0)

* **Conbench infrastructure and v2 proposal:** Discussion on the future of Arrow's benchmarking tool, including a proposal by Wes McKinney to rewrite the backend in Go for better stability and historical data management.
