* **Arrow 25.0.0 Release:** Voting is underway for version 25.0.0 containing over 220 resolved issues, with discussion regarding a potential RC2 or immediate patch release to address performance regressions in PyArrow's `to_pylist` for Spark. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%2025.0.0%20-%20RC1)

* **Conbench v2 Overhaul:** Proposal to modernize the Arrow benchmarking infrastructure by implementing a Go backend and a Svelte-based frontend to improve UI responsiveness and result storage performance. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Status%20of%20Arrow%20conbench%20data%20%2B%20conbench%20OSS%20project)

* **C++ Variant Type Support:** Coordination on implementing the Variant type in Arrow C++ to match the existing Rust implementation, aimed at benefiting downstream projects like Iceberg. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Variant%20Type%20Support)

* **JSON Representation of Schemas:** Discussion on establishing a canonical human-readable JSON format for Arrow schemas to facilitate easier use in REST APIs and ADBC metadata exchange. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Format%20JSON%20representation%20of%20Arrow%20schemas)

* **Deprecating Tensor IPC Messages:** Proposal to deprecate the experimental Tensor and SparseTensor IPC messages in favor of using fixed-shape and variable-shaped tensor extension types within standard RecordBatches. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20IPC%20Deprecate%20Tensor%20and%20SparseTensor%20messages%3F)

* **ADBC Extension for DuckDB:** Columnar has proposed donating its DuckDB ADBC extension to the Arrow project to provide a community-governed client for connecting DuckDB to various external databases. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Extension%20for%20DuckDB)

* **CI Resource Optimization:** Following a request from ASF Infrastructure, the project successfully reduced its GitHub Actions usage by approximately 27% to better share limited runner resources. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Consumption%20of%20ASF%20Shared%20GitHub-hosted%20Runner%20Resources)

* **Arrow Erlang Donation:** The Apache Arrow Erlang repository has been officially transferred to ASF infrastructure following a successful community vote and donation process. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RESULT%20VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **Stale Issue Triage:** A call for volunteers to help reproduce old bugs and close out irrelevant issues to reduce the growing maintenance burden on the core contributors. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Closing%20out%20old%20issues%20-%20extra%20help%20wanted)
