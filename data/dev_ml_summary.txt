* **IBM Db2 Support:** IBM engineers are working on adding Arrow support for Db2, initially by contributing a new ADBC driver and exploring future Flight SQL integration. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **ADBC Partitioned Bulk Ingest:** Proposal for a new API to support parallel data ingestion across distributed writers (like Spark) with atomic coordinator-level commit and abort semantics. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **UTF-8 Metadata Restriction:** Discussion on whether to strictly enforce UTF-8 for field and schema metadata to ensure compliance with Flatbuffers specifications and improve cross-language interoperability. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8?)

* **Rust Patch Releases:** A series of emergency patch releases (58.3.0, 57.3.1, 56.2.1) were issued for the Rust implementation to address potential offset overflow vulnerabilities in specific array types. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Planned%20patch%20releases%20this%20week)

* **Erlang Implementation Donation:** The community voted to accept the donation of an Erlang implementation of Arrow, which currently utilizes bindings to the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **Parquet Bloom Filter Performance:** Proposal to optimize Parquet Split Block Bloom Filter (SBBF) probes using branchless code that facilitates compiler autovectorization for AVX2 and NEON. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Flight SQL Dialect Metadata:** Proposal to add new SqlInfo codes for dialect-specific features, such as LIMIT/OFFSET syntax and NULL ordering, to help clients generate more efficient pushdown queries. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20dialect-related%20SqlInfo%20codes)

* **Arrow .NET 23.0.0:** The .NET implementation reached a new milestone with the 23.0.0 release, featuring an improved verification suite and a new PowerShell-based validation script. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20.NET%2023.0.0%20RC0)

* **Community Over Code Hackathon:** Planning is underway for an Arrow-focused hackathon at the Community Over Code event in Glasgow this October to address "needs champion" issues and documentation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Hackathon%20at%20Community%20Over%20Code%20Glasgow%202026%20%E2%80%94%20interest%20from%20Arrow?)
