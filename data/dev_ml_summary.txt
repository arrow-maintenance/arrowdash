* **Numba support for PyArrow:** Discussion on integrating Numba support into PyArrow to enable high-performance JIT compilation for Arrow arrays while managing ABI stability and maintenance burdens. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Python%20Numba%20support%20for%20PyArrow)

* **Java JDK 17 requirement:** Consensus was reached to move the minimum requirement for the Java implementation to JDK 17 starting with the 20.0.0 release. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Java%20Set%20JDK%2017%20minimum%20for%20Arrow%20Java%2020.0.0)

* **Donation of pyarrow-stubs:** The project approved the donation of the `pyarrow-stubs` codebase, which will be integrated into the main repository to provide official PEP 484 type annotations for PyArrow. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20PYTHON%20Donate%20pyarrow-stubs%20to%20Apache%20Arrow)

* **Apache Arrow 24.0.0 release:** Voting is underway for the 24.0.0 release candidate, which includes approximately 258 resolved issues across the project. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%2024.0.0%20-%20RC0)

* **ADBC 23 release:** Release 23 of the Arrow Database Connectivity (ADBC) libraries was approved and published, including significant bug fixes and improvements for multiple language bindings. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Release%20Apache%20Arrow%20ADBC%2023%20-%20RC1)

* **LargeMap format addition:** Proposal to add a `LargeMap` type to the Arrow specification to support 64-bit offsets for map arrays, addressing compatibility issues with engines like Polars. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20LargeMap%20format%20addition)

* **C Device Data Interface experimentality:** Developers debated whether the C Device Data Interface should remain experimental and discussed the potential for a non-device-specific async stream API. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Experimentality%20of%20Arrow%20C%20Device%20Data%20Interface)

* **IPC representation of extension types:** Investigation into an IPC serialization bug for dictionary-encoded extension types led to a proposal to add metadata support to the `DictionaryEncoding` table. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:IPC%20representation%20of%20dictionary-encoded%20extension%20types)

* **Flight SQL protocol addition:** A vote passed to add an `is_update` boolean field to the `ActionCreatePreparedStatementResult` to help clients determine the correct execution path for prepared statements. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Flight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Arrow Rust 58.1.0 release:** The Rust implementation version 58.1.0 was approved and published, alongside a new release of the Arrow Object Store crate. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20RUST%20Release%20Apache%20Arrow%20Rust%2058.1.0%20RC1)
