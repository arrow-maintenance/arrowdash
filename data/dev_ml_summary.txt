* **Erlang implementation donation:** The community voted to accept the donation of an Erlang implementation (initially based on Rust bindings), which has now cleared the IP clearance process. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **ADBC partitioned bulk ingest:** A new API proposal seeks to enable distributed writers like Spark and Flink to perform parallel, atomic bulk ingestion into databases and table formats like Iceberg. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **arrow.range extension type:** Discussion is underway for a canonical extension type to represent mathematical intervals (e.g., [0, 10)), facilitating interoperability between libraries like Pandas and Polars. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Parquet Bloom Filter optimization:** New AVX2 and autovectorized SIMD implementations for Split Block Bloom Filters provide significant performance improvements for analytical query probes. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter.cc)

* **Metadata UTF8 enforcement:** The community is considering strictly enforcing UTF8 for field and schema metadata to avoid silent serialization issues with the underlying Flatbuffers format. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Field%2Fschema%2Fcustom%20metadata%20restriction%20to%20UTF8%3F)

* **AI-assisted PR reviews:** To mitigate a shortage of human reviewers, developers are testing automated GitHub Copilot reviews to catch trivial issues before human intervention. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **Flight SQL update indicator:** A vote passed to add an `is_update` boolean to prepared statements, helping drivers distinguish between queries and DDL/DML updates without client-side heuristics. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Flight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Apache Arrow .NET 23.0.0 release:** The latest .NET implementation release is now available on NuGet, including updates to Flight and Flight SQL components. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ANNOUNCE%20Apache%20Arrow%20.NET%2023.0.0%20released)

* **IBM Db2 ADBC driver:** Proposal to add official support for IBM Db2, with initial work moving to a dedicated repository under the community-maintained `adbc-drivers` organization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20ADBC%20DB2%20Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)
