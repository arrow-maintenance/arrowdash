* **Erlang implementation donation:** The community voted to accept a donation of an Erlang implementation of Apache Arrow, which currently utilizes bindings to the Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Donate%20Apache%20Arrow%20Erlang)

* **ADBC Partitioned Bulk Ingest API:** Proposal to add a partitioned bulk ingest API to ADBC to support atomic, parallelized writes from distributed systems like Spark and Flink. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Parquet Bloom Filter SIMD optimizations:** Discussion and implementation of AVX2 and NEON optimizations for Parquet Split Block Bloom Filters using xsimd and branchless autovectorization. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20AVX2%20SBBF%20probe%20for%20parquetbloom_filter.cc)

* **Canonical range extension type:** Proposal to introduce `arrow.range` as a canonical extension type for representing bounded intervals, with discussion on handling per-value closedness flags. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Add%20arrow.range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Automated GitHub Copilot reviews:** Enabling automated GitHub Copilot code reviews on pull requests to help catch trivial issues and reduce the burden on human maintainers. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSS%20Auto%20GitHub%20Copilot%20review)

* **Metadata restriction to UTF-8:** Discussion on enforcing UTF-8 validation for field and schema metadata to ensure compliance with the underlying Flatbuffers IPC specification. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Discuss%20Fieldschema%20custom%20metadata%20restriction%20to%20UTF8?)

* **IBM Db2 support via ADBC:** Engineers from IBM proposed adding Apache Arrow support for Db2, starting with a driver in the community-maintained ADBC driver foundry. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:DISCUSSADBCDB2%20Proposal%20Adding%20Apache%20Arrow%20Support%20for%20IBM%20Db2)

* **Flight SQL prepared statement improvements:** Agreement to add an `is_update` field to prepared statements, allowing drivers to select the correct execution path for DDL/DML vs. queries. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:VOTE%20Flight%20SQL%20Add%20is_update%20field%20to%20ActionCreatePreparedStatementResult)

* **Rust implementation patch releases:** Coordination of multiple patch releases for versions 58.x, 57.x, and 56.x to backport fixes for potential offset overflow issues. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:RUST%20Planned%20patch%20releases%20this%20week)
