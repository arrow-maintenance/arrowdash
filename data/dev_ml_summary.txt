* **Erlang implementation donation:** Acceptance of a new Erlang implementation based on Rust bindings into the Apache Arrow project, including completed IP clearance and repository transfer. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Donate%20Apache%20Arrow%20Erlang)

* **Parquet SIMD Bloom Filter optimizations:** Proposal and implementation of AVX2 and branchless autovectorization for Parquet Bloom Filter probes, yielding 3-7x performance gains on C++ read paths. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:AVX2%20SBBF%20probe%20for%20parquet%2Fbloom_filter%2Ecc)

* **ADBC Partitioned Bulk Ingest API:** Proposal for a new write-side API in ADBC to enable distributed writers like Spark or Flink to perform parallel writes with coordinated atomic commits. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:ADBC%20Partitioned%20Bulk%20Ingest%20API)

* **Canonical extension type for ranges:** Discussion on a proposed `arrow.range` type for bounded intervals, focusing on how to support both fixed-schema and per-value boundary closedness. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20arrow%20range%20canonical%20extension%20type%20for%20bounded%20ranges)

* **Auto GitHub Copilot reviews:** Trial and automation of AI-powered pull request reviews to catch trivial issues early, with discussion on refining the workflow to reduce noise for draft PRs. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Auto%20GitHub%20Copilot%20review)

* **Conbench infrastructure redesign:** Discussion on modernizing the Conbench benchmarking project, including a potential rewrite of the backend in Go to improve stability and sustainability. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Status%20of%20Arrow%20conbench%20data%20%2B%20conbench%20OSS%20project)

* **Flight SQL dialect metadata:** Addition of new `SqlInfo` codes to help clients identify backend SQL dialects for better pushdown of LIMIT/OFFSET and literal types. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Add%20dialect-related%20SqlInfo%20codes)

* **Arrow Rust 59.0.0 release:** The community approved and released version 59.0.0 of the Apache Arrow Rust implementation. [Thread](https://lists.apache.org/list?dev@arrow.apache.org:lte=2M:Release%20Apache%20Arrow%20Rust%2059%2E0%2E0%20RC2)
